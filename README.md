# BookingAir

An application for booking airline tickets.

## About the Project

BookingAir is a service for searching and booking airline tickets.  
Built with Java using modern backend development technologies.

## Features

- Search flights by date, destination, price  
- Book selected tickets  
- Manage user bookings  
- Database integration for storing flights and bookings information  

## Technologies

- Java 17  
- Spring Boot  
- Spring Security  
- REST API  
- JWT (JSON Web Token)  
- MySQL  
- Hibernate / JPA  
- Docker  
- Maven  
- Lombok  
- Swagger / OpenAPI  
- Git
